  • Product Details

    ー Size ー

    RAMN Base Board Size :
    W550mm×D900mm×H110mm

    RAMN (Resistant Automotive Miniature Network) is a compact, hands-on automotive cybersecurity testbed, designed for secure development, testing, and training in vehicular networks.

    The system includes four interconnected Electronic Control Units (ECUs) built entirely on Printed Circuit Boards (PCBs). Each ECU can be programmed to generate realistic CAN/CAN-FD traffic and interact with other modules. RAMN can reproduce the same network behavior as PASTA, an established open-source automotive security project, enabling compatibility with existing research and training material.

    ー Key Features ー

    Security Testing Environment :
    Safely simulate cyberattacks, intrusion attempts, and defensive strategies on a real CAN/CAN-FD bus without risk to actual vehicles.
    Digital-Twin Integration :
    Connect seamlessly with the autonomous driving simulator CARLA. Events inside CARLA directly impact the physical CAN/CAN-FD network, and network manipulations feed back into the simulator, enabling realistic attack-and-defense-in-the-loop scenarios.
    Training & Education Tool :
    With its approachable size and interactive hardware controls, RAMN is ideal for teaching vehicle networking, penetration testing, and defensive programming.
    Plug-and-Play Setup :
    Powered by USB, recognized as a standard CAN adapter, and easily expandable with additional modules.

    ー Optional ECU Extensions ー

    ECU A
    CANCAN Bus Display :
    Features an integrated screen to visualize real-time CAN traffic, making it easier to observe, analyze, and demonstrate network behavior.
    ECU B
    Steering & Controls :
    Includes a rotating potentiometer (steering wheel), a two-position switch (parking brake), and a four-position rotary switch (lighting controls).
    ECU C
    Driver Inputs :
    Provides two sliding potentiometers (accelerator and brake pedals) and a joystick (gear stick). The joystick supports multiple inputs:

    - Up/Down → request gear changes
    - Left/Right → operate turn indicators
    - Center press → activate horn

    ECU D
    Engine & Indicators :
    Equipped with a three-position ignition key (OFF / ACC / IGN) and eight LEDs to represent vehicle status or sensor signals.
